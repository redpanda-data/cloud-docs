= Enable Redpanda SQL on a BYOVPC Cluster
:description: Provision SQL-specific cloud resources and enable Redpanda SQL on a BYOVPC cluster.
:page-layout: index

Unlike standard BYOC clusters, Redpanda does not create resources in your cloud account for BYOVPC clusters. To enable Redpanda SQL on a BYOVPC cluster, you must provision the SQL-specific cloud resources yourself and supply them to the cluster as customer-managed resources (CMRs). The required resources and Terraform module differ by cloud provider.

Select your cloud provider to get started.
