= pg_get_statisticsobjdef_columns
:description: The pg_get_statisticsobjdef_columns() function retrieves information about the columns associated with an extended statistics object.
:page-topic-type: reference

The link:https://www.postgresql.org/docs/current/functions-info.html#FUNCTIONS-INFO-CATALOG[`pg_get_statisticsobjdef_columns()`^] is a system catalog information function that retrieves information about the columns associated with an extended statistics object.

== Syntax

[source,text]
----
pg_get_statisticsobjdef_columns(statobj oid)
----

== Restrictions

* The `statobj` parameter is required.
