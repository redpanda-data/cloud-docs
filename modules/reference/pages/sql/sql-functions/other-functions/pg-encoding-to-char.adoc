The link:https://www.postgresql.org/docs/current/functions-info.html#FUNCTIONS-INFO-CATALOG[pg_encoding_to_char()] is a system catalog information function that converts an encoding internal identifier to a human-readable name.

== Syntax

The syntax for the `pg_encoding_to_char()` function is as follows:

[source,sql]
----
pg_encoding_to_char(number)
----

== Parameters

The following parameters are required to execute function:

* `number`: specifies the integer value representing the encoding identifier

== Examples

[source,sql]
----
SELECT pg_encoding_to_char(1);

 pg_encoding_to_char
---------------------
 EUC_JP
(1 row)
----

[source,sql]
----
SELECT pg_encoding_to_char(0);

 pg_encoding_to_char
---------------------
 SQL_ASCII
(1 row)
----

[source,sql]
----
SELECT pg_encoding_to_char(-1);

 pg_encoding_to_char
---------------------

(1 row)
----
