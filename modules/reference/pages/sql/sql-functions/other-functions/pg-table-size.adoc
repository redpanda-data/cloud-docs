= pg_table_size()
:description: The pg_table_size() function retrieves the size of a specific table, including its associated storage components but excluding indexes.
:page-topic-type: reference

link:https://www.postgresql.org/docs/current/functions-admin.html[`pg_table_size()`^] is a system administration function that retrieves the size of a specific table, including its associated storage components but excluding indexes.

== Syntax

[source,sql]
----
pg_table_size(regclass)
----

== Parameters

* `regclass`: Name or object identifier (OID) of the table.
