= pg_table_size()
:description: The link:https://www.postgresql.org/docs/current/functions-admin.html[pg_table_size()^] is a system administration function that retrieves the size of a specific table.
:page-topic-type: reference

The link:https://www.postgresql.org/docs/current/functions-admin.html[pg_table_size()^] is a system administration function that retrieves the size of a specific table, including its associated storage components but excluding indexes.

== Syntax

The syntax for the `pg_table_size()` function is:

[source,sql]
----
pg_table_size(regclass)
----

== Parameters

These parameters are required to execute this function:

* `regclass`: name or object identifier (OID) of the table whose size is to be retrieved
