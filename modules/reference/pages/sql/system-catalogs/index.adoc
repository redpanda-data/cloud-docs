= System Catalogs
:description: This section provides information about the system catalogs that are a foundational component of the Redpanda SQL database management system, serving as a repository for metadata about the database and its various objects.
:page-topic-type: reference

This section provides information about the system catalogs that are a foundational component of the Redpanda SQL database management system, serving as a repository for metadata about the database and its various objects. The information in this section is divided into groups according to the kind of operation they perform as follows:

* *xref:reference:sql/system-catalogs/catalogs/pg_description.adoc[System Catalogs].* Find out more about storing schema metadata. The system catalog tables include `pg_attrdef`, `pg_attribute`, `pg_class`, `pg_constraint`, `pg_depend`, `pg_description`, `pg_index`, `pg_namespace`, `pg_settings`, `pg_statio_user_tables`, and `pg_type`.
